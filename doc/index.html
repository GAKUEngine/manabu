⚙学 Manabu Client for GAKU Engine

The Manabu client is built to access the GAKU Engine API.

Usage

Testing

Testing requires a working Docker installation with docker-compose accessble by your user account. You can run rspec directly and a container will be brought up on port 9000. Each time you run rspec the container is brought up and down, so each run will take some time. You can save time when running specs often by bringing up the test container in advance with rake testing:container:up and waiting for about 10 seconds for the container to come up before running specs.
